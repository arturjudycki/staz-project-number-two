Cięcie projektu graficznego, wykonanego w Figmie, składającego się z:
* <b>Home Page</b>,
* <b>Login Page</b>.
* <b>Sign Up Page</b>.
* <b>FAQ Page</b>.

Użyte technologie:
* HTML, CSS, SASS (składnia .sass), JS, jQuery,

z zastosowaniem: 
* nomenklatury BEM,
* biblioteki Font Awesome,
* oraz podejściem RWD.
